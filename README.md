# ChonoTask

Una aplicación web para gestionar tareas utilizando la técnica Pomodoro. Maximiza tu productividad organizando el trabajo en intervalos de 25 minutos con descansos estratégicos.

## Características

- Gestión de tareas (crear, organizar, priorizar)
- Temporizador Pomodoro integrado
- Dashboard con estadísticas de productividad
- Autenticación segura
- Interfaz moderna e intuitiva

## Tecnologías

**Frontend**
- [Next.js 16](https://nextjs.org/) - Framework React
- [React 19](https://react.dev/) - UI library
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Estilos
- [shadcn/ui](https://ui.shadcn.com/) - Componentes
- [Recharts](https://recharts.org/) - Gráficos
- [Reactflow](https://reactflow.dev/) - Diagramas
- [dnd-kit](https://dndkit.com/) - Drag & drop

**Backend**
- [Next.js](https://nextjs.org/) - API routes
- [Prisma](https://www.prisma.io/) - ORM
- [PostgreSQL](https://www.postgresql.org/) - Base de datos (Neon)
- [NextAuth](https://next-auth.js.org/) - Autenticación

**Utilidades**
- [Zod](https://zod.dev/) - Validación de datos
- [Lucide React](https://lucide.dev/) - Iconos
- [date-fns](https://date-fns.org/) - Fechas
- [Cloudinary](https://cloudinary.com/) - Imágenes
